*** Création de la base de données ***

 (SGBD utilisé: MySQL)


- aller dans le dossier "Fichiers BDD"

- exécuter le script "database creation file.sql" (par exemple via phpMyAdmin)

pour créer la base de données "actuamboara"

- exécuter ensuite le script "actuamboara data.sql"

afin d'insérer quelques données dans la base





*** Lancement du programme ***


- s'assurer que le SGBD est en marche et écoute sur le port 3306

- double-cliquer sur le fichier "GestionActu.jar"

se trouvant dans le dossier du projet

- . utilisateur: amboara

  . mot de passe: '' (laisser vide)





*** Codes sources ***

- aller dans le dossier "Codes sources"

- le point d'entrée du programme est: src/mainprogram/GestionActu.java
